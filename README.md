# Estudo de MVC no PHP

**Estrutura de Pasta**

```bash
$ mvc
  +--app
    +--classes
    +--controller
    +--functions
    +--model
    +--traits
    +--view
  +--core
  +--public
    index.php
  bootstrap.php
  composer.php
  config.php
```

**Criando o Autoload**

$ composer dump-autoload -o

**Instalando o Twig**

$ composer require
